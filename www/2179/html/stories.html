It is also possible to list and search stories directly from the command line using the story command:

tmt story ls
tmt story show
tmt story show test/create

Current status of story coverage from implementation, testing and documentation point of view can be viewed using the coverage subcommand with optional regular expression for filtering:

tmt story coverage
tmt story coverage cli
tmt story coverage --implemented
